## Brief Description

ParikshaMitra is an online exam slot booking system developed to manage and allocate exam slots efficiently for students. The system allows students to select their preferred exam city, date, and time slot based on availability. It is designed to handle multiple users booking slots simultaneously without overbooking. The project uses Redis for atomic slot management and MongoDB for storing user and booking data. The system is built using the MERN stack (MongoDB, Express.js, React.js, Node.js) and ensures fair slot allocation, prevents double booking, and provides real-time slot availability.
